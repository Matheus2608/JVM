Detailed Description

Motor principal de decodificação (parsing) do arquivo Java.

Consome os bytes brutos lidos pela classe Leitor e os distribui rigorosamente de acordo com a Especificação Oficial da JVM (Java Virtual Machine Specification), preenchendo a estrutura 'class_info' (árvore de sintaxe) na memória do C++.
