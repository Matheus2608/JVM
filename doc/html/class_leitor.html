Detailed Description

Classe responsável pela leitura segura do arquivo binário (.class).

Encapsula a manipulação do stream do arquivo e resolve o problema de "Endianness" (a JVM utiliza a arquitetura Big-Endian, enquanto processadores modernos usam Little-Endian), fornecendo métodos que leem diretamente os bytes no formato correto.
